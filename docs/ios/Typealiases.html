Type Aliases

The following type aliases are available globally.

  • KlarnaPaymentAction

    PaymentAction is just a a type-alias to a String describing the action that is being performed. It’s provided in failure and warning delegate methods.

    Declaration

    Swift

    public typealias KlarnaPaymentAction = String
  • KlarnaPaymentMethodCategory

    PaymentMethodCategory is just a a type-alias to a String describing a payment method category. It can be extended by the Merchant at any time.
